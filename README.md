WebDriverIO
=============

Prerequisite
-----------------
-NodeJS is Installed
-Yarn is Installed

On First Clone, Run - Yarn install

To Run UI Tests
---------------------

yarn run ui --cucumberOpts.tagExpression='@Ui' (Can use any Tag)

To Run API Tests
---------------------

yarn run api --cucumberOpts.tagExpression='@Api' (Can use any Tag)
